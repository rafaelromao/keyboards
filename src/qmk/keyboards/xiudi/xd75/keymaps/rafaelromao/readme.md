## xd75/rafaelromao
### A pseudo split keyboard, in a XD75 body.

The keyboard in the picture below is a XD75, used to design and test this layout.

### QMK

#### Compile

`qmk compile -kb xd75 -km rafaelromao`

#### Flash

`qmk flash -kb xd75 -km rafaelromao`
